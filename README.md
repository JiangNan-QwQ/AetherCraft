### 如何使用?   
- 你可以在<a href=https://github.com/jiangnan-qwq/aethercraft>Github</a>或<a href=https://gitee.com/jiangnan-qwq/aethercraft>Gitee</a>找到本项目并克隆到 **/xnlr** 目录下并用`python xnlr/xnlr.py`启动脚本，还有一些依赖需要安装，脚本开发完毕后会在官网更新详细教程。
